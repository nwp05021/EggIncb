# IncubatorC3 Framework v0.3.0 (GrowBed Main UI Adopted)

- Reuses GrowBed-0.4.0 main-screen layout philosophy (header/body/footer, UTF-8 Korean font)
- PageStack UI engine kept for menus/edit/confirm
- Main screen supports multiple pages (rotate on main to change page)
- Bottom status bar shows: Heater / Motor / Fan / Humidifier

## Controls
- Main screen:
  - Rotate: change main page (P1..)
  - Click: Settings
- Settings:
  - Rotate: cursor
  - Click: edit
  - Long press: back
- Edit:
  - Rotate: change value (accel)
  - Click: save & back
  - Long press: cancel & back
